# Pmx2Fbx
使用前请仔细阅读并遵循模型附带的ReadMe/使用规则文档
Please carefully read and follow the model's ReadMe/usage rules document before use
使用前にモデルに含まれるReadMe/使用規則ドキュメントを必ずご確認ください,
사용 전에 모델에 포함된 ReadMe/사용 규칙 문서를 반드시 확인해 주세요


A plugin for Unity that can convert .pmx files to .fbx files , while retaining skeletal skinning, blend shapes, and material textures, and can also add VRChat adaptation to the .fbx files.
用于Unity的插件，可以一键将.pmx文件转换为.fbx文件，并保留骨骼蒙皮，形态键和材质贴图，同时还可以为.fbx文件添加Vrchat适配
