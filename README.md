# Pmx2Fbx
使用前请仔细阅读并遵循模型附带的ReadMe/使用规则文档

Please carefully read and follow the model's ReadMe/usage rules document before use

使用前にモデルに含まれるReadMe/使用規則ドキュメントを必ずご確認ください,

사용 전에 모델에 포함된 ReadMe/사용 규칙 문서를 반드시 확인해 주세요


A plugin for Unity that can convert .pmx files to .fbx files with one click, preserving bone skinning, shape keys, and material textures, and can also convert .fbx files to humanoid skeletons and add VRChat adaptation.


用于Unity的插件，可以一键将.pmx文件转换为.fbx文件，并保留骨骼蒙皮，形态键和材质贴图，同时还可以将.fbx文件转为人型骨骼，添加Vrchat适配
